# Pizza map

## Description
Map that shows all the pizza restaurants with detailed information in Tartu.<br>
LIVE DEMO: https://karlveskus.github.io/pizza-map/

## Setup
1. Clone repository and open project folder
2. Run ```npm run prod```
3. Open index.html in your browser
