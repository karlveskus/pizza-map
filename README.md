# Pizza map

## Description
Map that shows detailed information about pizza restaurants located in Tartu.<br>
LIVE DEMO: https://karlveskus.github.io/pizza-map/

- When the user clicks the marker, an info window opens with detailed information about this restaurant
- From menu, it's possible to filter restaurants by its name
- Responsive design makes web page render well on a variety of devices and screen sizes 

## Setup
1. Clone the repository
2. Change directory to project folder
3. Run ```npm install``` to get all dependencies
3. Run ```npm run prod``` to build required files
4. Open ```index.html``` in your browser

## Use your own Google and Foursquare API user data
- Google Maps API key and secret are located at ```index.html``` (link on line 23)
- Foursquare API client ID and secret are located at ```config.js```
